#### Hydra 2 TD

![](https://raw.githubusercontent.com/grinchdubs/grnch.xyz_photos/refs/heads/main/TD_Tools/Screenshot%202025-10-27%20112332.png)

Creates bidirectional communication between Hydra's live coding environment and TouchDesigner. This enables real-time pattern coding while integrating TouchDesigner's data processing, beneficial for generative artists seeking immediate visual feedback combined with robust node-based workflows.

Find it on [Github](https://github.com/grinchdubs/Hydra2TD-1.0)

[Documentation](https://github.com/grinchdubs/Hydra2TD-1.0/tree/main/documentation) can be found here


#### TD Copier Tools

![](https://raw.githubusercontent.com/grinchdubs/grnch.xyz_photos/refs/heads/main/TD_Tools/Screenshot%202025-10-29%20005020.png)

Automates the repetitive task of transferring parameter values between operators. The utilities handle copying settings across similar operators and duplicating entire chains with their configurations preserved, saving considerable time during complex network construction.

Find it on [Github](https://github.com/grinchdubs/TD_Copier_Tools)

[Documentation](https://github.com/grinchdubs/TD_Copier_Tools/blob/main/README.md) can be found here


#### PoST

![](https://raw.githubusercontent.com/grinchdubs/grnch.xyz_photos/refs/heads/main/TD_Tools/Screenshot%202025-10-27%20133548.png)

Incorporates professional color grading and finishing directly into TouchDesigner's environment. This eliminates the need to export work to external software like DaVinci Resolve or After Effects, keeping the entire pipeline contained within a single application — particularly valuable for live performance contexts.

Find it on [Github](https://github.com/grinchdubs/PoST)

[Documentation](https://github.com/grinchdubs/PoST/blob/main/README.md) can be found here
